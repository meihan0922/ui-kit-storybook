# Tabs

樣式相同，只有大小之分。

- `<Tabs>`：需放在最外層，用來建立 provider。
- `<TabsList>`： `<Tabs>` 的子元件，為上方列表，內容物必須為`<Tab>`。
- `<TabsPanels>`： `<Tabs>` 的子元件，為下方 body 顯示區塊，內容物必須為`<TabsPanel>`。
- `<Tab>`： `<TabsList>` 的子元件。
- `<TabsPanel>`： `<TabsPanels>`的子元件。
